# Status Overlay

This configuration uses a custom script to display a system status overlay without a permanent bar.

## Usage
Press **SUPER + S** to toggle the overlay.

## Content
The overlay shows:
- Time and Date
- Wi-Fi status
- Battery percentage
- VPN status

## Configuration
The script is located at `~/.config/hypr/scripts/status_overlay.sh`.
It uses `wofi` to display the menu. You can edit the script to change the content or the launcher.
