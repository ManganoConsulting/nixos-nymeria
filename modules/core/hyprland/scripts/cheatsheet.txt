# Hyprland Cheatsheet

## Windows
SUPER + C           Close Window
SUPER + V           Toggle Floating
SUPER + P           Pseudo Tiling
SUPER + J           Toggle Split
SUPER + Left/Right  Move Focus
SUPER + SHIFT + ... Move Window

## Workspaces
SUPER + 1..0        Switch Workspace
SUPER + SHIFT + ... Move Window to Workspace
SUPER + S           Toggle Special Workspace
SUPER + Mouse       Scroll Workspaces

## Apps
SUPER + Enter       Terminal (Ghostty)
SUPER + W           Warp Terminal
SUPER + E           File Manager
SUPER + R           App Launcher (Wofi)
SUPER + B           Browser (Vivaldi)

## System
SUPER + S           Status Overlay
SUPER + F1          This Cheatsheet
SUPER + ALT + L     Lock Screen
SUPER + M           Exit Hyprland

## Media
Vol Up/Down         Volume
Mute                Mute Audio
Brightness +/-      Screen Brightness
Media Keys          Play/Pause/Next/Prev

## Screenshots
SUPER + Print       Screenshot Region
SUPER + SHIFT + Prt Screenshot Full
SUPER + F9          Screen Record Toggle
